Features

The current features of BACMan v2.1 include:

  • Integrated management of BAC based physical mapping hybridization data including:
    • Gridding of library filters
    • Preparation of probes as genomic clones or development as overgos
    • Probe sequence information
    • Experimental data for hybridizations
  • Production of 'Code 39' Bar Code labels for autoradiographs
  • Creation of scoring grids for printing on transparency film
  • An experiment builder for pooled probe multiplex hybridization experiments
  • Deconvolution of pooled probe multiplex designs for up to 576 probes with triple redundancy
  • Experiment definition tool which allows user to define additional pooled probe multiplex experiment formats
  • Overview queries of the database including probe hybridization status and film scoring status
  • Multiple data entry interfaces including
    • Bar code assisted manual data entry using a programmable Zebex CCD bar code scanner
    • BACGrid data entry
    • OCR data entry using ABBYY Fine Reader in conjunction with the BACEater data processing application
  • Identification of films that possibly contain scoring errors and may need to be error checked
  • Printing error checking sheets

FULL BACMAN SUITE - This self installing package includes all of the following packages as well as the BACMan manual.
  • BACMan - Main BACMan MS Access database. This includes an example dataset and requires a local copy of MS Access to run.
  • BACGrid - A MS Word macro for entering scored film data from scoring templates.
  • BACEater - Converts scored template data from ABBYY FineReader into the format used for BACMan.
  • Barcode Font - The bar-coding capabilities of BACMan rely on a free Code 39 barcode font. This font must be installed in your windows font directory.
  • BACMan Web Package - includes the MyBACMan.pl Perl script that creates the tables needed for web publication of BACMan as well as the php pages for presenting data on the web

The use of this full suite of packages is described in the BACMan User Manual.
